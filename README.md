# ⚔️ BattleMon Online

> Real-time turn-based creature battles.  
> Built for fun. Built to win.

---

## 🎮 What is this?

BattleMon Online is a multiplayer web game where players battle each other using custom creatures.

- ⚡ Fast-paced turn-based combat  
- 🔥 Unique creature system  
- 🌐 Online multiplayer (Socket.io)  

---

## 🧬 Game Basics

- 🧍‍♂️ 1v1 battles  
- 🧠 Strategy based
- 🐾 3 creatures per team  

### Types
`Fire` · `Water` · `Grass` · `Electric` · `Earth` · `Normal`

### Stats
`HP` · `ATK` · `DEF` · `SPD`

---

## 🛠️ Tech Stack

**Backend**
