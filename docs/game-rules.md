# Game Rules

## Battle Format
- 1v1 turn-based battle
- Each player has a team of 3 creatures
- One active creature at a time

---

## Turn Order
- Determined by Speed (SPD)
- Higher SPD moves first

---

## Priority System
- Quick Hit always goes first
- If both use priority moves → compare SPD

---

## Damage System
- Based on:
  - Attacker ATK
  - Defender DEF
  - Move Power

---

## STAB (Same Type Attack Bonus)
- If move type = creature type
- Damage multiplier: 1.5x

---

## Accuracy
- Each move has accuracy %
- If roll fails → move misses

---

## Buff Moves
- Focus Boost → ATK ×1.5
- Guard Up → DEF ×1.5
- Speed Up → SPD ×1.5

---

## Healing
- Recover restores 50% max HP

---

## Win Condition
- A player wins when opponent has no creatures left alive
